## 4. Presentación del proyecto

El proyecto que he hecho este año es una **aplicación de gestión de restaurante** en Java.

La idea es que el encargado o el camarero de un restaurante pueda gestionar los clientes, ver qué mesas están libres y consultar la carta, todo desde una aplicación de consola conectada a una base de datos real.

Lo he hecho porque muchos restaurantes pequeños todavía usan papel o Excel para estas cosas y una aplicación así les facilitaría el trabajo.

**Tecnologías que he usado:**
- Java para programar la aplicación
- PostgreSQL como base de datos
- JDBC para conectar Java con la base de datos
- XML y XSD para exportar y validar datos de la carta
- Git y GitHub para el control de versiones

**Lo que más me ha costado:** conectar Java con PostgreSQL al principio porque el driver no lo encontraba y tardé un rato en solucionarlo.

**Lo que mejor me ha salido:** el diseño de la base de datos. El diagrama E/R me quedó bien y las consultas SQL funcionan correctamente.
